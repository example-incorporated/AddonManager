# AddonManager
A plugin of Enmity (Custom Discord Client for iOS) to manage addons easily.

## Installation
https://raw.githubusercontent.com/m4fn3/AddonManager/master/dist/AddonManager.js

## Usage
Use /addon command or open plugin setting page.
